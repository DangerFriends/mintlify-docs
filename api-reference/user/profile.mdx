---
title: 'Get Profile'
openapi: 'GET /api/profile'
---

Get the authenticated user's profile data including uploads, votes, and statistics.

## Request

<CodeGroup>

```bash cURL
curl "https://api.fucksornot.com/api/profile?type=counts" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.fucksornot.com/api/profile?type=counts', {
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});
```

</CodeGroup>

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `type` | string | Data to retrieve: `uploads`, `votes`, or `counts` |

## Response

### With `type=counts`

```json
{
  "counts": {
    "uploads": 42,
    "votes": 156
  }
}
```

### With `type=uploads`

```json
{
  "uploads": [
    {
      "id": "upload-uuid",
      "filename": "image.webp",
      "description": "My upload",
      "upvotes": 10,
      "downvote_count": 2,
      "created_at": "2025-01-24T12:00:00Z"
    }
  ]
}
```

### With `type=votes`

```json
{
  "votes": [
    {
      "id": "vote-uuid",
      "upload_id": "upload-uuid",
      "vote": "fucks",
      "created_at": "2025-01-24T12:00:00Z"
    }
  ]
}
```

## Caching

This endpoint is cached for **30 seconds** with private cache control.

## Use Cases

- Display user dashboard
- Show activity statistics
- List user's content and votes
