## List All Uploads

Get a paginated list of all uploads including private ones.

### Request

```bash
curl "https://api.fucksornot.com/api/admin/uploads?page=1&limit=20&search=gadget" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | 1 | Page number |
| `limit` | integer | 20 | Items per page |
| `search` | string | - | Search by description |

### Response

```json
{
  "uploads": [
    {
      "id": "upload-uuid",
      "user_id": "user-uuid",
      "filename": "image.webp",
      "description": "Cool gadget",
      "upload_type": "image",
      "upvotes": 42,
      "downvote_count": 5,
      "is_private": 0,
      "created_at": "2025-01-24T12:00:00Z",
      "username": "johndoe"
    }
  ],
  "page": 1,
  "limit": 20,
  "total": 1500
}
```

## Delete Upload

Delete any upload.

### Request

```bash
curl -X DELETE https://api.fucksornot.com/api/admin/uploads/UPLOAD_ID \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Response

```json
{
  "success": true
}
```

---

## Remove Tag from Upload

Remove a specific tag from an upload.

### Request

```bash
curl -X DELETE https://api.fucksornot.com/api/admin/uploads/UPLOAD_ID/tags/TAG_ID \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Response

```json
{
  "success": true
}
```

---

## Delete Tag

Delete a tag entirely (removes from all uploads).

### Request

```bash
curl -X DELETE https://api.fucksornot.com/api/admin/tags/TAG_ID \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Response

```json
{
  "success": true
}
```

<Warning>
  This removes the tag from all uploads that use it.
</Warning>

## List All Tags

Get all tags with admin context.

### Request

```bash
curl "https://api.fucksornot.com/api/admin/tags?page=1&limit=50" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
