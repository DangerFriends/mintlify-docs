## List Users

Get a paginated list of all users.

### Request

```bash
curl "https://api.fon.app/api/admin/users?page=1&limit=20&search=john" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | 1 | Page number |
| `limit` | integer | 20 | Items per page |
| `search` | string | - | Search by username or email |

### Response

```json
{
  "users": [
    {
      "id": "user-uuid",
      "username": "johndoe",
      "email": "john@example.com",
      "is_admin": false,
      "is_shadow_banned": false,
      "has_eleven_access": false,
      "created_at": "2025-01-24T00:00:00Z",
      "upload_count": 42,
      "vote_count": 156
    }
  ],
  "page": 1,
  "limit": 20,
  "total": 500
}
```

## Delete User

Delete a user account.

### Request

```bash
curl -X DELETE https://api.fon.app/api/admin/users/USER_ID \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Response

```json
{
  "success": true
}
```

<Warning>
  This permanently deletes the user and all their data including uploads and votes.
</Warning>

## Toggle Admin Status

Grant or revoke admin privileges.

### Request

```bash
curl -X POST https://api.fon.app/api/admin/users/USER_ID/admin \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"isAdmin": true}'
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `isAdmin` | boolean | `true` to grant, `false` to revoke |

### Response

```json
{
  "success": true
}
```

### Request

```bash
curl -X POST https://api.fon.app/api/admin/users/USER_ID/eleven-access \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \

```

## Force MFA

Force enable or disable MFA for a user.

### Request

```bash
curl -X POST https://api.fon.app/api/admin/users/USER_ID/mfa \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"enabled": false}'
```

<Note>
  Use this to disable MFA for a user who has lost access to their authenticator.
</Note>
