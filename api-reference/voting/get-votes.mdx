---
title: 'Get Votes'
openapi: 'GET /api/vote'
---

Get all votes cast by the authenticated user.

## Request

<CodeGroup>

```bash cURL
curl https://api.fucksornot.com/api/vote \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.fucksornot.com/api/vote', {
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});
const { votes } = await response.json();
```

</CodeGroup>

## Response

```json
{
  "votes": [
    {
      "id": "vote-uuid-1",
      "user_id": "user-uuid",
      "upload_id": "upload-uuid-1",
      "vote": "fucks",
      "created_at": "2025-01-24T12:00:00Z"
    },
    {
      "id": "vote-uuid-2",
      "user_id": "user-uuid",
      "upload_id": "upload-uuid-2",
      "vote": "does_not_fuck",
      "created_at": "2025-01-24T11:00:00Z"
    }
  ]
}
```

## Vote Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | UUID | Vote identifier |
| `user_id` | UUID | Your user ID |
| `upload_id` | UUID | Upload that was voted on |
| `vote` | string | `fucks` or `does_not_fuck` |
| `created_at` | timestamp | When the vote was cast |

## Use Cases

- Display which uploads the user has voted on
- Show vote status in UI
- Prevent duplicate vote attempts
